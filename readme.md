` Less Extend ` is A Less Extention For Faster Efficient Develop.




## Quick Start


#### How to use ?


```console
npm install --save-dev less-extend
```


#### If Failed in Webpack ~

```console
npm install --save-dev less-loader less
```


#### Import Less File !

```css
@import '~less-extend/index.less'
```
